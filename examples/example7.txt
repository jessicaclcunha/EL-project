SQuery -> Query NUM ListaIds SEP ListaLinhas

Query -> SELECT Colunas FROM ID

Colunas -> STAR | ListaColunas

ListaColunas -> ListaColunas COMMA ID | ID

ListaIds -> ListaIds ID | ID

ListaLinhas -> ListaLinhas SEP ListaVal | ListaVal

ListaVal -> ListaVal Coluna | Coluna

Coluna -> COLUNANUM | COLUNASTR


SELECT    = /SELECT/
FROM      = /FROM/
NUM       = /\d+/
STAR      = /\*/
COMMA     = /,/
SEP       = /\|/
COLUNANUM = /\d+(\.\d+)?/
COLUNASTR = /'[^']*'/
ID        = /[a-zA-Z][a-zA-Z0-9_]*/
